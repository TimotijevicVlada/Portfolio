About

-Who I am-

I am Vladimir and I am Web Developer

I was born in 1990 in Belgrade. I attended the elementary school "Uzicka Republika" in New Belgade. Since I have always loved martial arts, when I turned 7 I started to train karate in my school. I practiced karate for the next 18 years with a lot of success in comepitions.

When I finished primary school, I enroll in a "Sports High School". When I graduated from high school, I decided to enroll in the "Faculty of Sport and Phisical Education" in Belgrade. After 4 years, I obtained a bachelor's degree and emmediately after that I enrolled in master's studies at the same faculty. After 1 year I graduated the master's degree.

After my studies, I start working in the karate club and at the same time I work in kindergartens with children from 3 to 6 years. After some time, I get the opportunity to work in one of the largest gyms in Belgrade, "Sportska Akademija Kocovic" where I have been working for 5 years.

At the end of the 2020, I realize that I need bigger challenges at work and after few months I enrolled in the "ComTrade Code academy" programming school. Before I started attending the Code academy, I started educating myself form the internet and realized that I actually love doing it. After 5 and a half months I graduated the ComTrade Code course. During that time I made a couple of projects that were posted on my Git hub account. After finishing Code course I continue to educate myself in the same direction and that will be my next jurney.

HTML

HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images.

CSS

Cascading style sheets are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

JavaScript

JavaScript is a dynamic programming language that's used for web development, in web applications, for game development, and lots more. It allows you to implement dynamic features on web pages that cannot be done with only HTML and CSS.

React

React (also known as React. js or ReactJS) is a free and open-source front-end JavaScript library for building user interfaces or UI components.
